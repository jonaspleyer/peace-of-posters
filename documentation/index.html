      Documentation #

      peace-of-posters consists of two components that complement each other. The first component, provides boxes which display content on the specified page. Secondly, their styles are customized by themes which can be preconfigured and reused when creating new posters.

      Note #

      The package is still in early stages and thus might undergo major changes in the future.
